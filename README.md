# simple-calculator
 Simple Python Calculator

This is a beginner-friendly calculator program written in Python. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division based on user input.

 Features

Takes two numbers as input from the user

Allows the user to choose an operation:

Addition (+)

Subtraction (-)

Multiplication (*)

Division (/)

Displays the result instantly

Handles division by zero error

 Technologies Used

Python (Basic conditional statements and user input)

 How to Run

Make sure Python is installed.

Run the program using:

python calculator.py


Enter the numbers and select the operation when prompted.

 What I Learned

Using input() function

Working with conditional statements (if, elif, else)

Performing arithmetic operations

Handling basic errors

 Purpose

This project was created to practice Python fundamentals and understand how decision-making works in programming.
